## 📌 About The Project
**Chester Editor Image** adalah asisten visual berbasis AI yang dirancang untuk **mendeteksi, menghapus, merekonstruksi, dan menyempurnakan elemen visual pada gambar secara instan**.

Sistem ini mampu menghilangkan gangguan visual seperti:
- objek yang tidak diinginkan
- orang di background
- latar belakang berantakan
- noise visual
- tepi objek kasar
- shadow yang hilang setelah editing

Dengan bantuan AI, hasil akhir dibuat **halus, realistis, dan natural seolah elemen tersebut tidak pernah ada sebelumnya**.

> ✨ Fokus utama project ini adalah *AI-powered image restoration, smart object manipulation, and creative visual enhancement*.

# 🌟 Core Features
## 🎯 22 Fitur Unggulan Chester Editor Image

### 🖼️ Image Processing & AI Core
- ✅ **1. Upload Image + Background Remove + AI Mode**  
  Upload gambar dengan sistem drag-and-drop modern, lalu gunakan AI model untuk proses remove background dan optimasi kualitas.

- ✅ **2. Shadow Reconstruction**  
  Menambahkan atau memperbaiki bayangan objek agar hasil edit terlihat realistis dan tidak melayang.

- ✅ **3. Manual Erase & Restore Brush + Zoom**  
  Pengguna dapat menghapus atau mengembalikan area tertentu secara manual menggunakan brush presisi tinggi, lengkap dengan fitur zoom.

- ✅ **4. Edge Smoothing**  
  Menghaluskan tepi objek setelah background removal agar terlihat natural.

- ✅ **5. Export PNG / JPG / JPEG / WEBP**  
  Mendukung export ke berbagai format gambar populer.

- ⚠️ **6. Expand Canvas / Lebarkan Gambar**  
  Memperluas area canvas untuk memberi ruang tambahan pada desain.  
  *Status: masih dalam tahap optimasi.*

---

### 🎯 Smart Selection & Object Editing
- ✅ **7. Auto Select & Object Removal**  
  AI secara otomatis mendeteksi objek hanya dengan satu klik, lalu objek dapat langsung dihapus.

- ✅ **8. Object Delete + Undo / Redo System**  
  Setiap perubahan tersimpan dalam history stack sehingga pengguna bebas membatalkan atau mengulang proses editing.

- ✅ **9. Add Elements / Stickers**  
  Menambahkan elemen visual seperti stiker, icon, atau ornamen untuk desain kreatif.

- ⚠️ **10. Merge Multiple Images**  
  Fitur penggabungan beberapa gambar ke dalam satu canvas.  
  *Status: masih tahap pengembangan.*

---

### 🎨 Quality Enhancement & Layout
- ✅ **11. Resolution Upgrade / Upscale**  
  Menampilkan serta meningkatkan resolusi gambar menjadi kualitas lebih tinggi (HD / Ultra HD).

- ✅ **12. Text to Image**  
  Menghasilkan gambar otomatis dari prompt teks menggunakan AI generator.

- ✅ **13. Canva-like Positioning System**  
  Sistem drag, resize, layer movement, dan penempatan elemen yang intuitif seperti Canva.

- ⚠️ **14. Position Lock Feature**  
  Mengunci posisi objek agar tidak bergeser saat proses editing.  
  *Status: masih perlu penyempurnaan.*

- ⚠️ **15. Image Animation**  
  Memberikan animasi ringan pada elemen gambar agar lebih hidup.

### 🪄 Advanced Editing Tools
- ✅ **16. Flip & Rotate**  
  Membalik gambar horizontal/vertical dan rotasi sesuai kebutuhan desain.

- ✅ **17. AI Beauty Retouch**  
  Menyesuaikan skin smoothing, brightness, contrast, saturation, dan sharpness otomatis.

- ✅ **18. Smart Background Replacement**  
  Mengganti latar belakang dengan warna custom atau gambar lain tanpa mengubah objek utama.

- ✅ **19. Color Filters**  
  Menambahkan filter artistik seperti grayscale, warm tone, sepia, cinematic, dan efek visual lainnya.

- ✅ **20. Photo to Short Video**  
  Mengubah gambar statis menjadi video pendek dengan animasi transisi.

- ✅ **21. AI Prompt Edit**  
  Mengedit gambar hanya menggunakan instruksi teks, misalnya:  
  *“hapus orang di belakang dan ganti langit jadi sunset.”*

- ✅ **22. Brush Selection Tool**  
  Seleksi manual area spesifik menggunakan brush masking.
